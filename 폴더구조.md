app/ → Remix 애플리케이션 루트
--routes/ → 각 라우트 파일(home.tsx, 동적 라우트 등)
--components/ → 재사용 가능한 UI 컴포넌트
--layouts/ → 페이지 레이아웃 컴포넌트
--hooks/ → 커스텀 훅
--store/ → Redux Toolkit slice, store 설정(store/index.ts, slices/…)
--services/ → API 통신 로직, 외부 서비스 래퍼
--utils/ → 공통 유틸 함수
--styles/ → 전역/모듈 CSS 또는 Tailwind 확장 (globals.css, tailwind.css 등)
--assets/ → 이미지, 아이콘, 폰트 등 정적 리소스
public/ → 정적 파일(파비콘, Open Graph 이미지 등)
config/ → 설정 파일 모음(tailwind.config.ts, vite.config.ts 분리 시)
tests/ → 유닛/통합 테스트 (원하면 app 내부에 **tests** 형태로 배치 가능)
